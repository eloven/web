## Material-start

    这是一个比较完整的angular项目实例。包含常用的第三方组件库，数据存储，用户认证，路由缓存等功能。
    
#### Material使用指南
    
    1.全局安装 node > 8.0 和 angular/cli
    2.克隆本项目并安装依赖
    git clone
    npm install
    3.启动服务
    ng serve
    4.查看地址
    http://localhost:4200
    
#### 模块划分说明

    UI模块       Material
    远程数据库    Filebase
    表格工具      handsontable
