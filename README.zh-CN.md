### 主要模块

```bash
   
   ng-client   // angular客户端
   
   nest-server  // nest.js node 服务端
  
```

### dev

> client port:4200
> server port:9001

```bash

   git clone

   cd client

   npm install

   npm run start

   cd nest-server

   npm install

   npm run start

```

### about client
